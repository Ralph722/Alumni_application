================================================================================
                            HOW TO ACCESS ADMIN DASHBOARD
================================================================================

STEP 1: Run the Flutter app
        $ flutter run

STEP 2: Login to the application

STEP 3: Navigate to the main navigation screen

STEP 4: Long-press on the Profile icon (bottom right)
        ⚠️ Important: LONG-PRESS, not tap!

STEP 5: Admin Dashboard will load

STEP 6: To exit, click the back button (floating action button)
